# TODOApp

A vibrant and energetic Android To-Do List application built with modern standards.

## Description
TODOApp is a robust task management application that strictly follows **Material Design 3 (M3)** guidelines. It features a high-energy color palette and a fully **adaptive 2-pane layout**, providing an optimized user experience across phones, tablets, and foldables.

## Features
- **Core Task Management**: Effortlessly create, edit, toggle completion, and delete tasks.
- **Adaptive UI**: Automatically switches between a single-pane list and a side-by-side list-detail view based on screen size using `ListDetailPaneScaffold`.
- **Modern Navigation**: Powered by **Navigation 3**, providing a reactive, state-driven navigation architecture.
- **Reactive Data Layer**: Built with **Room Database** and Kotlin `Flow` for real-time synchronization between data and UI.
- **Material 3 Vibrant Design**: Features a dynamic color scheme, expressive typography, and an adaptive app icon.
- **Edge-to-Edge**: Full edge-to-edge display support for a modern, immersive feel.

## Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Navigation**: Navigation 3 (androidx.navigation3)
- **Adaptive Layout**: Compose Material Adaptive (androidx.compose.material3.adaptive)
- **Database**: Room
- **Concurrency**: Coroutines & Flow
- **Theme**: Material Design 3
