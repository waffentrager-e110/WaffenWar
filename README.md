# WaffenWar

Пошаговая стратегия на тему Второй Мировой войны.

## Особенности
- Низкие требования к устройству
- Игра не требует подключения к интернету
- Активная разработка игры гарантирует частого добавления нового контента
- Сложная система боев обеспечивает интересный геймплей

## Ресурсы
